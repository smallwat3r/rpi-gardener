PIN ASSIGNMENTS
===============

RPi GPIO Header (J1)
--------------------
Pin 1  (3V3)     -> Power rail for 3.3V sensors
Pin 2  (5V)      -> Power for LCD
Pin 3  (GPIO2)   -> I2C SDA (OLED, LCD)
Pin 4  (5V)      -> Power
Pin 5  (GPIO3)   -> I2C SCL (OLED, LCD)
Pin 6  (GND)     -> Ground
Pin 11 (GPIO17)  -> DHT22 Data
Pin 9,14,20,25,30,34,39 -> GND

OLED RPi (J2) - SSD1306 128x64
------------------------------
Pin 1: VCC (3.3V)
Pin 2: GND
Pin 3: SDA (GPIO2)
Pin 4: SCL (GPIO3)

LCD I2C (J3) - 1602A with PCF8574
---------------------------------
Pin 1: VCC (5V)
Pin 2: GND
Pin 3: SDA (GPIO2)
Pin 4: SCL (GPIO3)

DHT22 (J4)
----------
Pin 1: VCC (3.3V)
Pin 2: DATA (GPIO17)
Pin 3: GND

Pico (U1)
---------
Pin 1  (GP0)  -> OLED SDA
Pin 2  (GP1)  -> OLED SCL
Pin 31 (GP26) -> Moisture Sensor 1
Pin 32 (GP27) -> Moisture Sensor 2
Pin 34 (GP28) -> Moisture Sensor 3
Pin 36 (3V3)  -> Power for Pico peripherals

OLED Pico (J5) - SSD1306 128x64
-------------------------------
Pin 1: VCC (Pico 3.3V)
Pin 2: GND
Pin 3: SDA (GP0)
Pin 4: SCL (GP1)

Moisture Sensors (J6, J7, J8)
-----------------------------
Pin 1: VCC (Pico 3.3V)
Pin 2: Signal (GP26/GP27/GP28)
Pin 3: GND


I2C BUS NOTES
=============
Both OLED (RPi) and LCD share the same I2C bus on GPIO2/GPIO3.
- OLED modules typically use address 0x3C
- LCD backpacks use 0x27 or 0x3F
Check for address conflicts if devices don't respond.


PCB DIMENSIONS
==============
- Total board size: 65mm x 56mm (standard RPi HAT size)
- Compact design with Pico integrated on same footprint
- Pico USB port extends slightly past right edge for access
- Mounting holes: M2.5 at standard HAT positions (3.5mm from edges)
- 2-layer board, 1.6mm thickness recommended


ASSEMBLY NOTES
==============
1. Solder the 40-pin header (J1) first - ensure alignment with RPi
2. Solder pin headers for connectors (J2-J8)
3. Solder the Pico module last (or use pin headers for removability)
4. Optional: Use female headers for Pico to allow removal


USB PASSTHROUGH
===============
The Pico's USB port is accessible for direct connection to the RPi.
Use a USB-A to Micro-USB cable from any RPi USB port to the Pico.
This provides:
- Serial communication for moisture sensor data
- Programming/firmware updates for the Pico
